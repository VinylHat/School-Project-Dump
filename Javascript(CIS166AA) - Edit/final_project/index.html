Final Project Theme

I always struggle with more open assignments, I am not the creative type. I decided, I like video games, I spend most of my free time with video game. Making a few, simple, games comes more naturally then a store page. I dont feel like making this an uphill battle, so thats what I am sticking with.

I hope you enjoy something in here, because this was tedious.

Simon says is a classic that every one should be familiar with. This version is pretty hard as the pattern is new every round, (It makes the code more interesting). Just a fair warning, if a color is lit for longer than usual, that means it was used multiple times in a row.

Blackjack, also known as 21, is a very simple card game. Again, not trying to reinvent the wheel here. The dealer is a little dumb, so don't go too hard.

Mole is whack-a-mole, another simple concept that I thought would be easy. I looked into seeing if you could use vectors in JS/HTML, and saw that it is possible with jquery. Perfect.

Rock, Paper, Scissors is a very basic concept that came to mind first when picking a topic because of how easy it is to implement.
